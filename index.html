Software Skills

AutoCAD

A powerful drafting software used for creating precise 2D drawings and 3D models. Widely used in architectural, civil, and structural design documentation.

Autodesk Revit

A Building Information Modeling (BIM) software for designing, modeling, and coordinating architecture, structure, and MEP systems in a unified environment.

Navisworks

A project review and clash detection tool used for integrating multidisciplinary BIM models. Helps identify conflicts and streamline coordination before construction.

Dynamo

A visual programming tool for automating tasks and generating complex geometry in Revit. Enhances design flexibility and workflow efficiency.

BIM 360

A cloud-based construction management platform that supports real-time collaboration, document sharing, and issue tracking across all project phases.

Photoshop

A professional image editing software used for creating presentations, enhancing visuals, and producing high-quality architectural renderings.

LUMION

A real-time 3D rendering software used for creating high-quality architectural visualizations. Ideal for producing realistic images, animations, and walkthroughs quickly and easily.

MS Office

A suite of productivity tools including Word, Excel, and PowerPoint, essential for documentation, data analysis, and professional reporting.
